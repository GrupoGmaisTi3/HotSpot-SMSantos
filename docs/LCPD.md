## 📝 Como Implementar

### 1. **Personalize os dados da empresa:**
Substitua todos os campos entre colchetes `[ ]`:
- Nome da empresa/provedor
- CNPJ
- Endereço completo
- Contatos (telefone, e-mail)
- Dados do DPO (Encarregado)

### 2. **Integre ao fluxo de cadastro:**
```html
<!-- Em register.html, adicione antes do formulário -->
<a href="terms.html" target="_blank" class="terms-link">
  📋 Ler Termos de Uso e Privacidade
</a>

<!-- Checkbox obrigatório -->
<div class="form-group">
  <label>
    <input type="checkbox" required id="acceptTerms">
    Li e concordo com os <a href="terms.html" target="_blank">Termos de Uso e Política de Privacidade</a> *
  </label>
</div>
```

### 3. **Armazene o consentimento:**
No `api/create-user.php`, adicione:
```php
// Registrar consentimento LGPD
$consent = [
    'cpf' => $cpf,
    'accepted_at' => date('Y-m-d H:i:s'),
    'ip_address' => $_SERVER['REMOTE_ADDR'],
    'user_agent' => $_SERVER['HTTP_USER_AGENT'],
    'version' => '2026.05.29',
    'terms_hash' => hash_file('sha256', __DIR__ . '/../terms.html')
];

// Salvar em banco de dados ou arquivo
file_put_contents(
    __DIR__ . '/../consents/' . $cpf . '.json',
    json_encode($consent, JSON_PRETTY_PRINT)
);

// Ou enviar para o MikroTik como comment adicional
$comment .= "|Consent:" . date('Y-m-d');
```

### 4. **Crie pasta para logs de consentimento:**
```bash
mkdir /var/www/hotspot/consents
chmod 750 /var/www/hotspot/consents
```
